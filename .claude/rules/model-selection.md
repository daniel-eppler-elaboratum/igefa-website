# Modell-Auswahl

Empfehle proaktiv das passende Modell, wenn die Aufgabe klar ist:

| Aufgabe | Modell | Begründung |
|---------|--------|-----------|
| Komplexe Architektur, großes Coding, Analysen | **Opus 4.6** | Höchste Reasoning-Qualität |
| Content, Prompt Engineering, kreative Aufgaben | **Sonnet 4.6** | Bestes Preis-Leistungs-Verhältnis |
| E-Mails, einfache Formatierungen, schnelle Fragen | **Haiku 4.5** | Schnell und günstig |

Wechsel mit `/model` in der Claude Code Session.
Wenn die aktuelle Aufgabe ein anderes Modell erfordern würde, weise den Nutzer proaktiv darauf hin.
